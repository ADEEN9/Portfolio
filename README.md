# Ameenudeen A - Personal Portfolio

A personal portfolio website showcasing the skills, projects, and professional background of Ameenudeen A, a Software Developer.

## Overview

This is a responsive vertical scrolling single-page portfolio built using standard modern web technologies. It highlights personal info, technical expertise, past projects, and includes a functional contact form.

## Technologies Used

- **HTML5** & **CSS3**
- **JavaScript (ES6+)**
- **FontAwesome** & **BoxIcons** for icons
- **EmailJS** for handling contact form submissions

## Setup

Since this is a static website, no build step or package manager is required. 

1. Clone the repository.
2. Open `index.html` in your web browser, or use an extension like VS Code's **Live Server** to preview the site locally.

## Features

- Fully responsive design (Desktop & Mobile navigation)
- Light / Dark theme toggle
- Contact form tied to EmailJS
- Direct link to download resume
- Project showcase section
